 \section{实验目的}
\begin{itemize}

\end{itemize}

 \section{实验内容}
 本实验通过运行shell命令查看系统信息以达到了解openEuler操作系统的目的。另外运行简单的C程序和汇编程序以了解基于ARMv8-64的开发环境。
 
 主要包含以下的内容：
 \begin{enumerate}
 	\item 查看系统信息;
 	\item 测试编程环境;

 	\item 编译和运行汇编程序；
 \end{enumerate}
